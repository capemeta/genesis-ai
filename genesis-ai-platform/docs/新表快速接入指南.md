## 快速开始（3 种方式）

### 方式 1：完全自动化（推荐，最简单）

**只需 1 行代码！**

#### 步骤 1：创建模型

```python
# models/knowledge_base.py
from sqlalchemy import String, Text
from sqlalchemy.orm import Mapped, mapped_column
from models.base import Base, AuditMixin
from uuid import UUID

class KnowledgeBase(Base, AuditMixin):
    """
    知识库模型
    
    继承 AuditMixin 自动获得审计字段：
    - owner_id, created_by_id, updated_by_id
    - created_by_name, updated_by_name
    - created_at, updated_at
    """
    __tablename__ = "knowledge_bases"
    
    # 🔍 配置可搜索字段（用于 search 参数）
    __searchable_fields__ = ["name", "description"]
    
    # 租户隔离（必需）
    tenant_id: Mapped[UUID] = mapped_column(nullable=False, index=True)
    
    # 业务字段
    name: Mapped[str] = mapped_column(String(255), nullable=False)
    description: Mapped[str | None] = mapped_column(Text, nullable=True)
    visibility: Mapped[str] = mapped_column(String(20), default="private")
    
    # 软删除（可选）
    deleted_at: Mapped[datetime | None] = mapped_column(DateTime, nullable=True)
```

**重要说明**：
- `__searchable_fields__`：配置可搜索的字段，用于前端 `search` 参数的模糊搜索
- `AuditMixin`：自动包含审计字段（owner_id, created_by_id, updated_by_id 等）
- `tenant_id`：必需，用于租户隔离

#### 步骤 2：注册到 CRUD 工厂（1 行代码）

#### 三种过滤模式

**1. 简单搜索（search）** - 模糊匹配
```python
# 后端：配置搜索字段
class KnowledgeBase(Base):
    __searchable_fields__ = ["name", "description"]

# 前端：使用 search 参数
fetchKnowledgeBases({ search: "测试" })
```

**2. 精确过滤（filters）** - 精确匹配
```typescript
// 前端：使用 filters 参数
fetchKnowledgeBases({ 
  filters: { 
    owner_id: "uuid-xxx", 
    visibility: "private" 
  } 
})
```

**3. 高级过滤（advanced_filters）** - 支持操作符
```typescript
// 前端：使用 advanced_filters 参数
fetchKnowledgeBases({
  advanced_filters: [
    { field: "name", op: "like", value: "测试" },
    { field: "created_at", op: "gte", value: "2024-01-01" }
  ]
})
```

**支持的操作符**：`eq`, `ne`, `gt`, `gte`, `lt`, `lte`, `like`, `in`, `not_in`, `is_null`, `is_not_null`

#### 参数兼容性
后端 `ListRequest` 自动兼顾了 **Refine 标准** 与 **内部标准**，你可以自由选择：

- **Refine 标准**: `_start`, `_end`, `_sort`, `_order`, `q`
- **内部标准**: `page`, `page_size`, `sort_by`, `sort_order`, `search`

**详细说明**：请参考 [doc/增删改查指南.md](../doc/增删改查指南.md) 的"过滤与搜索"章节

from fastapi import APIRouter, Depends, UploadFile
from uuid import UUID
from core.security import get_current_active_user
from core.response import auto_refine_response

# 获取自动生成的 router
from api.crud_registry import crud_factory
router = crud_factory.get_router("KnowledgeBase")

# 添加自定义路由
@router.post("/{kb_id}/documents")
@auto_refine_response

    kb_id: UUID,

#### API 客户端

```typescript
// src/lib/api/tag.ts
import axiosInstance from './axios-instance'

/**
 * 查询标签列表
 * 
 * 使用 GET 方法，参数通过查询字符串传递
 */
export const fetchTags = async (
  params: TagListRequest = {}
): Promise<PaginatedResponse<Tag>> => {
  const response = await axiosInstance.get<ApiResponse<PaginatedResponse<Tag>>>(
    '/api/v1/tags/list',
    {
      params: {
        page: params.page || 1,
        page_size: params.page_size || 20,
        search: params.search || undefined,
        sort_by: params.sort_by || 'created_at',
        sort_order: params.sort_order || 'desc',
      }
    }
  )
  return response.data.data
}

/**
 * 创建标签
 * 
 * 使用 POST 方法，参数通过请求体传递
 */
export const createTag = async (
  data: { name: string; color?: string; description?: string }
): Promise<Tag> => {
  const response = await axiosInstance.post<ApiResponse<Tag>>(
    '/api/v1/tags/create',
    data
  )
  return response.data.data
}
```

1. 创建数据库表（SQL）

**自动获得**：
- 完整 CRUD API（5 个路由）
- 统一响应格式（包含 code、message、data 三字段，列表响应的 data 字段包含分页信息）
- 租户隔离、权限检查
- 审计字段、软删除
- **分页、过滤、排序（支持 3 种过滤模式）**
- 错误处理

**三种过滤模式**：
1. **简单搜索（search）**：模糊匹配，用于搜索框

**后续只需关注**：
