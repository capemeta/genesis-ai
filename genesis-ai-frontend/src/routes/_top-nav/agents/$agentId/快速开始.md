# Agent Builder 快速开始指南

## 🎯 添加节点的两种方式

### 方式一：拖拽到画布 ⭐ 推荐
1. 在左侧组件面板找到需要的组件
2. 按住鼠标左键拖拽组件
3. 移动到画布上的目标位置
4. 释放鼠标完成添加

**优势**: 精确控制位置，适合细致的布局

### 方式二：双击快速添加
1. 在左侧组件面板找到需要的组件
2. 双击组件卡片
3. 节点自动添加到画布中心

**优势**: 快速添加，适合快速搭建原型

## 🔗 连接节点

### 如何连接
1. **找到连接手柄**: 节点左右两侧的蓝色圆点（始终可见）
2. **开始连接**: 从源节点**右侧**的圆点开始拖拽
3. **完成连接**: 拖动到目标节点**左侧**的圆点并释放

### 连接提示
- ✅ 圆点始终可见，不需要悬停
- ✅ 悬停时圆点会放大，方便点击
- ✅ 拖拽时会显示连接线预览
- ✅ 可以删除连接线（点击选中后按 Delete）

## ⚙️ 配置节点

1. 点击画布上的节点
2. 右侧自动弹出配置面板
3. 修改节点名称、描述和参数
4. 点击"保存配置"完成

## ⌨️ 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| **拖拽组件** | 添加节点到指定位置 |
| **双击组件** | 添加节点到画布中心 |
| **Delete** | 删除选中的节点 |
| **Ctrl+Z** | 撤销上一步操作 |
| **Ctrl+Y** | 重做操作 |
| **Ctrl+D** | 复制选中的节点 |
| **Shift+拖拽** | 框选多个节点 |
| **滚轮** | 缩放画布 |
| **空格+拖拽** | 平移画布 |

## 🐛 常见问题

### Q: 看不到连接手柄？
A: 连接手柄（蓝色圆点）始终显示在节点左右两侧，如果看不到请：
- 检查节点是否在画布可视区域内
- 尝试缩放画布查看
- 确保浏览器支持最新的 CSS 特性

### Q: 如何快速添加多个节点？
A: 使用双击功能：
1. 双击左侧组件添加到中心
2. 拖动节点到合适位置
3. 重复以上步骤

### Q: 如何删除连接线？
A: 两种方式：
1. 点击连接线选中 → 按 Delete 键
2. 点击连接线 → 使用顶部快捷操作面板的删除按钮

### Q: 节点太小看不清？
A: 使用鼠标滚轮放大画布，或点击左下角的 + 按钮

## 💡 最佳实践

1. **从输入开始**: 先添加"用户查询"等输入节点
2. **逐步连接**: 按照数据流向依次添加和连接节点
3. **及时配置**: 添加节点后立即配置参数
4. **善用撤销**: 不确定时可以尝试，随时可以撤销
5. **保持整洁**: 定期整理节点位置，保持工作流清晰

## 🎨 视觉提示

- **紫色**: 输入触发器（用户查询、Webhook）
- **琥珀色**: 知识库相关（知识库检索、文档搜索）
- **蓝色**: AI 和逻辑（LLM、条件分支、工具）
- **绿色**: 输出节点（文本响应）
