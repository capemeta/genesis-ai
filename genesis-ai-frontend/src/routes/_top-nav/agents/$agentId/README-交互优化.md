# Agent Builder 交互优化说明

## 🎯 核心交互改进

### 1. **灵活的节点添加方式** ✨

#### 方式一：拖拽添加（推荐）
- **操作**: 从左侧组件面板拖拽组件到画布任意位置
- **视觉反馈**: 
  - 拖拽时显示自定义预览
  - 画布背景高亮显示
  - 抓手光标提示
- **优势**: 精确控制节点位置

#### 方式二：双击添加
- **操作**: 双击左侧组件卡片
- **效果**: 自动添加到画布中心位置
- **视觉反馈**: Toast 提示成功添加
- **优势**: 快速添加，适合快速原型

### 2. **键盘快捷键** ⌨️
| 快捷键 | 功能 |
|--------|------|
| `Delete` / `Backspace` | 删除选中的节点 |
| `Ctrl+Z` / `Cmd+Z` | 撤销操作 |
| `Ctrl+Y` / `Cmd+Y` | 重做操作 |
| `Ctrl+Shift+Z` | 重做操作（备用） |
| `Ctrl+D` / `Cmd+D` | 复制选中的节点 |
| `Shift` + 点击 | 多选节点 |
| `Shift` + 拖拽 | 框选多个节点 |

### 3. **撤销/重做系统** 🔄
- 完整的操作历史记录
- 顶部工具栏显示撤销/重做按钮
- 自动保存每次操作状态
- 按钮状态动态更新（无历史时禁用）

### 4. **增强的节点操作** 📦

#### 节点外观升级
- 更大的节点尺寸（280px）
- 圆角卡片设计（rounded-xl）
- 选中时有缩放效果和光环
- 图标带有彩色背景
- 底部显示配置提示

#### 节点连接
- **连接手柄始终可见**: 节点左右两侧的圆点
- **手柄样式**: 
  - 蓝色边框，带光晕效果
  - 悬停时放大并加粗
  - 清晰的视觉反馈
- **连接操作**:
  1. 从源节点右侧的圆点拖拽
  2. 拖动到目标节点左侧的圆点
  3. 释放鼠标完成连接
- **连接线特性**:
  - 平滑的曲线路径（smoothstep）
  - 悬停时高亮显示
  - 支持动画效果
  - 可直接点击删除

#### 快速操作面板
- 选中节点时，顶部中央显示快速操作栏
- 包含"复制"和"删除"按钮
- 一键完成常用操作

### 5. **组件搜索** 🔍
- 左侧面板顶部搜索框
- 实时过滤组件
- 支持搜索名称和描述
- 未找到时显示提示信息

### 6. **视觉反馈优化** 👁️

#### 画布信息
- 左上角显示节点数量和连接数量
- 实时更新统计信息
- 彩色圆点指示器

#### 空状态引导
- 画布为空时显示欢迎卡片
- 清晰的操作指引
- 视觉引导用户开始构建

#### Toast 通知
- 所有操作都有即时反馈
- 添加节点、删除、复制等都有提示
- 显示具体的操作信息

### 7. **多选和批量操作** 📑
- `Shift + 点击` 多选节点
- `Shift + 拖拽` 框选多个节点
- 批量删除选中的节点
- 显示选中节点数量

### 8. **改进的节点配置面板** ⚙️

#### 布局优化
- 更宽的面板（320px）
- 更清晰的分组
- 优化的表单控件
- 实时预览变量标签

#### 通用配置
- 所有节点都支持编辑名称和描述
- 统一的配置界面
- 保存和取消操作

### 9. **ReactFlow 原生功能** 🎮

#### 画布操作
- **平移**: 鼠标拖拽画布
- **缩放**: 鼠标滚轮或控件按钮
- **适应视图**: 自动调整显示所有节点
- **小地图**: 右下角缩略图导航

#### 连接功能
- 从节点手柄拖拽创建连接
- 悬停显示可连接的目标
- 自动路径规划
- 支持删除连接线

### 10. **响应式设计** 📱
- 适配不同屏幕尺寸
- 控件和小地图自动调整
- 移动端优化（如适用）

## 🎨 设计改进

### 节点设计
- 更现代的卡片式设计
- 彩色顶部条标识节点类型
- 图标背景带有渐变色
- 选中状态有明显视觉反馈

### 颜色方案
- **紫色**: 输入触发器

### 动画效果
- 平滑的过渡动画
- 连接线流动效果
- 按钮悬停反馈
- 节点选中缩放

## 🚀 性能优化

- 使用 `useCallback` 优化回调函数
- `useMemo` 缓存过滤结果
- 事件处理优化
- 减少不必要的重渲染

## 📝 使用建议

### 快速上手
1. **添加第一个节点**: 
   - 拖拽左侧"用户查询"到画布，或双击快速添加
2. **连接节点**: 
   - 从节点右侧的蓝色圆点拖拽到下一个节点的左侧圆点
   - 圆点始终可见，悬停时会放大
3. **配置节点**: 
   - 点击节点后，右侧面板显示详细配置
   - 可修改名称、描述和特定参数
4. **使用快捷键**: 
   - 熟悉 `Ctrl+Z/Y`、`Delete` 等快捷键
5. **善用撤销**: 
   - 不怕犯错，随时可以撤销重做

### 高效技巧
- 💡 **双击组件**: 快速添加到画布中心
- 💡 **Shift + 拖拽**: 框选多个节点
- 💡 **Ctrl+D**: 快速复制选中节点
- 💡 **空格 + 拖拽**: 平移画布视图
- 💡 **滚轮**: 缩放画布
